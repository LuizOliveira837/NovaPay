# NovaPay

Landing page de uma fintech fictícia, feita como atividade da FIAP.

## Objetivo

**Atividade - Muito estilo sem sofrimento e Rede Social dos Desenvolvedores**

Você deverá criar apenas 1 tela relacionada ao seu projeto Fintech.
Mantenha o HTML separado do CSS em arquivos .html e .css, respectivamente.
Não é necessário o uso de JavaScript.
A tela precisa estar relacionada ao seu projeto Fintech!
O projeto deve possuir responsividade para dispositivos mobile.
Utilize Tailwind CSS para construção do layout e estilização da interface.
Porém, garanta que o projeto abra na máquina do professor através do arquivo
.HTML. Projetos que não abrirem serão desconsiderados.

## Tecnologias

- HTML5
- CSS
- Tailwind CSS (via CDN)

## Como executar

Basta abrir o arquivo `index.html` no navegador. Não precisa de Node, npm nem
build.
